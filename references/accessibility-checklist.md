## DCL UI 접근성 (키보드 및 레이아웃)

DCL 대화상자는 단축키를 통해 완전히 조작 가능해야 하며 직관적인 흐름을 유지해야 합니다.

- [ ] **단축키(Mnemonic Shortcuts) 지정**: 사용자가 마우스 없이도 대화상자의 각 요소로 신속하게 이동할 수 있도록 모든 에디트 박스, 리스트 박스, 버튼에 앤드 기호 `&`를 붙여 키보드 단축키(`Alt + [지정 문자]`)를 제공합니다.
  - **DCL 예시**: `label = "레이어 &Add";` (Alt+A 입력 시 해당 컨트롤 클릭/포커스 실행).
- [ ] **탭 키 탐색(Tab Navigation) 순서**: 대화상자 탐색이 상식적인 흐름(왼쪽에서 오른쪽으로, 위에서 아래로)을 따르는지 점검하십시오. 행(row)과 열(column) 및 프레임 박스를 조화롭게 구성하여 탭 이동 순서가 꼬이지 않게 관리합니다.
- [ ] **기본 동작(Default Action)**: 양식을 쉽게 제출할 수 있도록 가장 주요한 버튼(예: 확인 버튼)에는 `is_default = true;` 속성을 부여하여 `Enter` 입력 시 실행되게 합니다.
- [ ] **취소 동작(Cancel Action)**: 창을 안전하게 닫을 수 있도록 취소 버튼에 `is_cancel = true;` 속성을 명시하여 `Esc` 입력 시 대화상자를 즉시 닫도록 제어합니다.
- [ ] **컨트롤 라벨 매핑**: 입력용 상자(edit box, list box 등)에는 항상 명확한 텍스트 설명을 짝지으십시오. 이름이 없는 공백 입력란을 방치해서는 안 됩니다.
- [ ] **적응형 크기 제어**: 사용자의 화면 DPI 설정이나 폰트 설정이 다르더라도 대화상자가 깨지지 않도록 가로 너비와 높이를 유연하게 조율합니다.

## CAD 도면 가독성 및 색상 대비

다양한 인쇄 장비(플로터)와 작업자의 캐드 화면 설정(어두운 배경 vs 밝은 배경)에 구애받지 않고 식별되어야 합니다.

- [ ] **어두운 배경 화면에서의 가독성**: CAD 뷰포트의 기본 검정 배경화면에서 어두운 파란색(ACI Color 5) 또는 어두운 회색(ACI Color 9)과 같이 시인성이 극도로 나쁜 색상을 중요한 외곽선이나 도면 정보 요소에 지정하는 일을 방지합니다.
- [ ] **밝은 종이 영역(Layout Sheets)에서의 가독성**: 흰색 레이아웃 도면 영역이나 실제 종이 출력물에서 식별하기 힘든 노란색(ACI Color 2)이나 하늘색(ACI Color 4) 글자 및 치수선을 배치하는 일을 방지합니다.
- [ ] **도면선 위계(Layer Hierarchy)**: 도면의 시각적 접근성을 높이기 위해 선가중치(선 굵기)와 레이어의 레이아웃 위계를 엄밀하게 적용합니다 (예: 외곽선은 굵게, 마감선/치수선은 얇게 조율).
- [ ] **선종류 구분(Linetype Scale)**: 숨은선(Dashed), 중심선(Center), 점선(Dotted) 등이 연속선(Continuous)과 확연히 비교될 수 있도록 알맞은 선종류 축척(`LTSCALE` 및 `CELTSCALE`)을 계산하여 적용합니다.

## 텍스트 및 치수 크기 축척 표준

- [ ] **축척 독립형 텍스트 지정**: 모든 주석, 텍스트 스타일 및 지시선 속성은 캐드의 주석 축척(Annotative scale)을 사용하거나 뷰포트 축척 비율에 연동하여 크기가 자동으로 비례 조정되도록 설계합니다.
- [ ] **최소 인쇄 글자 높이**: 출력물 종이 규격에서 글자를 한눈에 쉽게 읽을 수 있도록 실제 인쇄되었을 때의 최소 텍스트 크기를 2.0mm 이상(기본 주기 사항은 2.5mm 이상 유지 권장)으로 구성합니다.
- [ ] **치수 정밀도와 겹침 방지**: 치수선 간격, 보조선 오프셋, 화살표 크기가 도면 형상을 침범하거나 겹쳐서 도면을 판독하는 데 장애를 주지 않는지 검증합니다.

## 대표적인 DCL 접근성 적용 패턴

### 1. 단축키 및 기본/취소 속성이 포함된 표준 버튼 구성
```dcl
ok_button : button {
  key = "accept";
  label = "&OK";          // Alt+O 입력 시 동작 실행
  is_default = true;      // 엔터 키 입력 시 자동 제출
  width = 12;
}

cancel_btn : button {
  key = "cancel";
  label = "&Cancel";      // Alt+C 입력 시 동작 실행
  is_cancel = true;       // Esc 키 입력 시 창 닫기
  width = 12;
}
```

### 2. 단축키가 포함된 에디트 박스 구성
```dcl
: edit_box {
  key = "layer_name";
  label = "대상 레이어 이름(&L):"; // Alt+L 입력 시 이 텍스트 박스로 마우스 포커스 이동
  edit_width = 25;
}
```
